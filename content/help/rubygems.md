## 上游仓库

[https://rubygems.org](https://rubygems.org)

## 使用方法

添加新仓库地址

```
$ gem sources --add https://mirrors.tongji.edu.cn/nexus/repository/rubygems/
```

移除默认仓库

```
$ gem sources --remove https://rubygems.org/
```

清除本地缓存

```
$ gem sources -c
```
